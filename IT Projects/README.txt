# ITWS Personal Website

This project is a multi-page personal website developed for my Introduction to Information Technology and Web Science (ITWS) course at Rensselaer Polytechnic Institute. The site includes a homepage, resume page, and lab-based components, all connected through a consistent navigation system and unified styling.

## Overview

The goal of this project was to design and build a structured, user-friendly website from scratch while applying core web development concepts. The site emphasizes clean layout, readability, and consistency across pages.



## Technologies Used

- HTML for page structure and content organization  
- CSS for styling, layout, and visual consistency  
- JavaScript and jQuery for interactivity  
- GitHub for version control and project management  
- Microsoft Azure for deployment and hosting  



## Features

- Multi-page website with consistent navigation across all pages  
- Resume page structured with clear sections and readable formatting  
- Responsive layout with a focus on simplicity and usability  
- Organized file structure for scalability and maintainability  
- Interactive elements implemented using JavaScript and jQuery  


## Development Focus

Throughout this project, I focused on:
- Building a clean and consistent user interface  
- Structuring content in a clear and professional format  
- Improving navigation and overall user experience  
- Managing files and project updates through version control  
- Deploying and maintaining a live website using Azure
