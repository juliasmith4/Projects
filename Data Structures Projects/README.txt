# Data Structures Projects

This repository contains a collection of projects completed for a Data Structures course, focused on implementing core data structures and applying them to real-world style problems using C++.




## Overview

These projects emphasize problem solving, data organization, and working with structured and unstructured data. Many of the assignments involve reading input from files (JSON and text), filtering and processing that data, and generating meaningful output.

A major focus throughout this work was learning how to efficiently extract, organize, and utilize data to support different applications.




## Assignment Reference

Project specifications were provided through course materials:
https://github.com/jidongxiao/CSCI1200-DataStructures/tree/master/hws



All implementations, logic, and solutions in this repository were developed independently.



## Technologies Used

- C++
- Standard Template Library (STL): maps, sets, vectors
- File input/output (I/O)
- JSON and text file parsing

## Core Skills Demonstrated

- Data parsing and extraction from JSON and text files  
- Data filtering and transformation based on conditions  
- Organizing large datasets using maps and sets  
- Efficient data lookup and retrieval  
- Algorithm design and debugging  
- Recursive problem solving  
- Generating structured output from processed data  




## Included Projects

### YouTube Comment Analyzer
- Processed and filtered comment data from input files  
- Extracted relevant information and organized it for analysis  
- Demonstrated handling of large datasets and text-based input  

### Dating App Matching System
- Built a system to match users based on given criteria  
- Filtered and compared structured data to generate matches  
- Focused on logical conditions and efficient pairing  

### Ride Share Matching System
- Matched drivers and riders using input data  
- Implemented filtering logic to optimize pairings  
- Emphasized data organization and real-time style matching  

### Spotify Playlist Generator
- Processed song data to generate playlists based on conditions  
- Filtered and organized data to meet user-defined preferences  
- Demonstrated structured data handling and sorting  

### Yelp Business Recommendation System
- Analyzed business data and filtered results based on criteria  
- Generated recommendations using structured input data  
- Focused on efficient querying and result generation  

### Web Search Engine
- Built a search engine using maps and sets to create an inverted index  
- Parsed input data and ranked results using keyword density and backlinks  
- Implemented recursive crawling and data organization  
- Debugged ranking logic to improve accuracy and relevance  



## Input and Output

- Input files include JSON files and plain text files used to simulate real-world datasets  
- Programs parse and process this data to extract relevant information  
- Output files are generated to display results such as matches, rankings, and recommendations  



## Development Focus

Throughout these projects, I focused on:

- Writing clean and modular C++ code  
- Breaking down complex problems into manageable components  
- Building systems that rely on accurate data filtering and utilization  
- Debugging and refining logic for correctness and efficiency  
- Working with real-world style data formats and structures
